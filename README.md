# Expo project bootstrap

Reusable agent prompt for scaffolding new Expo React Native apps from a reference architecture.

## Usage

1. Open [PROJECT_BOOTSTRAP.md](./PROJECT_BOOTSTRAP.md)
2. Copy the prompt block into a new agent chat
3. Fill in the placeholders (`<REFERENCE_REPO_URL>`, `<APP_NAME>`, Figma links, etc.)

## Reference implementation

The default reference template is [expo-photo-gallery](https://github.com/hosam-hubspire/expo-photo-gallery) — Expo SDK 56, Uniwind, Bun, Biome, design-token pipeline, and optional Storybook / GraphQL / icons.

You can point **Reference repo (template)** at any repo that follows the same conventions.
